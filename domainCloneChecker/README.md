## Getting started
* --WINDOWS--
* Open VsCode
* Open terminal within VsCode
* `python --version`
* `python -m venv .venv`
* `.venv\scripts\activate `
* `pip3 install -r requirements.txt`


* --LINUX--
* Open Terminal
* Change to program path
* `python3 --version`
* `python3 -m venv .venv`
* `. .venv/bin/activate`
* `pip3 install -r requirements.txt`


To save new packages:
* `pip freeze > requirements.txt`
